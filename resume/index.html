实习生项目

标签管理系统

使用腾讯公司内部 PHP 框架实现的标签管理系统，在标签系统的内部做了分类处理，提供了极大的扩展性。接入了 ElasticSearch 实现了腾讯新闻，腾讯视频，腾讯网等一系列平台的标签管理，可以根据一个标签查找对应的文章，视频等资源。

快速管理组件

为腾讯公司内部的 PHP 框架添加的一个组件，使用了 BootStrap 和 jQuery 实现了快速建立类似 MyAdmin 的管理界面的功能，后端开发人员不需要写 HTML 和 JavaScript，就可以轻松建立给新闻和网站编辑使用的管理界面，同时提供了很大的扩展性。

实时数据统计

通过接入 Kafka 消息系统，使用 Spark 进行大数据分析，统计了前端 HTML5 界面的 PV UV 地域 平台 设备 时间点 等信息。

开源项目

使用全 JavaScript 开发的 Web 应用。后端使用 Node.js Express 4 MongoDB。前端使用 Vue Vuex Vue-router Vue-resource，并使用 Webpack Babel 进行全自动构建和测试，整个前端就是一个单页应用，目前正在完善中。由我负责所有的开发和测试部署工作。测试网址。代码开源在 wecanstuidio-site 。

ElasticYii(2016.7)

MDPreference(2016.3)
